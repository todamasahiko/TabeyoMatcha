# TabeyoMatcha

## サイト概要
抹茶に関する情報交換・共有サイト兼SNS

### サイトテーマ
「抹茶で楽しく、幸せに」

### テーマを選んだ理由
抹茶を通じて、楽しく幸せな気持ちになれたことがあった。同じように好きな人同士のコミュニティーの場があれば素敵だと感じたから。

### ターゲットユーザー
①老若男女問わず、抹茶が好きなユーザー
②抹茶が苦手、もしくはまだ食べたことのないユーザー

### 主な利用シーン
①食事前の下調べ
②話題作り
③贈り物前の品定め
